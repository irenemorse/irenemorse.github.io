## Course Overview

Data visualization is a function of both science and art. The computer revolution has generated a virtually infinite universe of data points on any topic one would care to study. As budding professionals and researchers we all need to learn how to harness information to tell stories about the world and to synthesize, summarize, and communicate what we care about and know about to others. Using the statistical software program R (and RStudio), we will learn to read, tame, and tidy data so that we can analyze and visualize our data artfully and effectively.

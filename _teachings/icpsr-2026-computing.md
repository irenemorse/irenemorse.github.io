## Course Overview

The course lectures introduce core concepts in data file construction and management, including useful computer skills facilitating the creation of data files for analysis in statistical programs. The lectures initially explore issues arising in datafile creation and management, then finishes with an introduction to using Stata, SPSS, and SAS.

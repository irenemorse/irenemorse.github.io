## Course Overview

This is a class designed to introduce you to the basics of LaTeX, a typesetting software to make research documents look professional and well-structured. LaTeX is incredibly useful for creating professional regression tables, formatting articles, and making nice presentations. By the end of this course, you should be able to:

• Know the basic structure and logic of the LaTeX programming language

• Understand the basic flow of the online Overleaf editor

• Create documents with LaTeX

• Format documents in LaTeX

• Use LaTeX to include figures and tables in a document

• Include references using BibTeX
